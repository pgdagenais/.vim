# Installation

```bash
git clone git@github.com:pgdagenais/vim-config.git
mv vim-config/ ~/.vim/
cd .vim/
git submodule update --init
```

## ChromeOS Installation with Chromebrew

```bash
cd /usr/local/share/vim/
git clone git@github.com:pgdagenais/vim-config.git
ln -s /usr/local/share/vim/vim-config/ ~/.vim/
cd ~/.vim/
git submodule update --init
```
